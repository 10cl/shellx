## Development Environment Setup

Follow these steps to set up your development environment for ShellX.ai:

**Step 1**: Clone the repository

<CodeGroup>

```bash Git
git clone https://github.com/10cl/shellx.git
cd shellx
```

</CodeGroup>

**Step 2**: Install dependencies

<CodeGroup>

```bash npm
npm install
```

```bash yarn
yarn install
```

</CodeGroup>

**Step 3**: Configure environment variables

Create a `.env` file in the root directory:

```env
# AI Service Configuration
AI_API_KEY=your_ai_api_key_here
AI_SERVICE_URL=https://api.your-ai-service.com

# Database Configuration
DATABASE_URL=your_database_url_here

# Security
JWT_SECRET=your_jwt_secret_here
```

**Step 4**: Start the development server

```bash
npm run dev
```

The development server will be available at `http://localhost:3000`.

## Architecture Overview

ShellX.ai follows a modular architecture designed for scalability and maintainability:

### Core Components

- **AI Engine**: Handles natural language processing and command interpretation
- **Terminal Interface**: Web-based terminal emulator with AI integration
- **Command Parser**: Intelligent command parsing and execution
- **Session Manager**: Manages user sessions and AI context
- **Security Layer**: Authentication and authorization system

### Technology Stack

- **Frontend**: React.js with TypeScript
- **Backend**: Node.js with Express
- **AI Integration**: OpenAI API / Custom AI models
- **Database**: PostgreSQL / MongoDB
- **Real-time Communication**: WebSockets
- **Mobile**: React Native for Android

## AI Integration

### Custom AI Agents

Create custom AI agents for specific tasks:

```javascript
// Example: Custom AI agent for file operations
class FileOperationAgent {
  constructor(apiKey) {
    this.apiKey = apiKey;
  }

  async processCommand(naturalLanguageInput) {
    // AI processing logic
    const command = await this.translateToCommand(naturalLanguageInput);
    return this.executeCommand(command);
  }

  async translateToCommand(input) {
    // AI translation logic
    return aiService.translate(input);
  }
}
```

### AI Model Configuration

Configure AI models in `config/ai.js`:

```javascript
module.exports = {
  models: {
    commandParser: {
      model: 'gpt-3.5-turbo',
      temperature: 0.3,
      maxTokens: 500
    },
    codeGenerator: {
      model: 'codex',
      temperature: 0.1,
      maxTokens: 1000
    }
  }
};
```

## API Integration

### REST API Endpoints

ShellX.ai provides RESTful APIs for integration:

```javascript
// Authentication
POST /api/auth/login
POST /api/auth/register

// Terminal Operations
GET /api/terminal/sessions
POST /api/terminal/execute
WS /api/terminal/ws

// AI Services
POST /api/ai/process
GET /api/ai/suggestions
```

### WebSocket Integration

Real-time terminal communication:

```javascript
const socket = io('ws://localhost:3000');

socket.on('terminal-output', (data) => {
  console.log('Terminal output:', data);
});

socket.emit('terminal-input', {
  command: 'ls -la',
  sessionId: 'session-123'
});
```
