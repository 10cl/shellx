## Welcome to ShellX.ai

ShellX.ai is an AI Agent Powered & Fast web-based, Android terminal that transforms your device into an AI-powered computing experience. Follow this guide to get started quickly.

### Installation Options

<AccordionGroup>
  <Accordion icon="globe" title="Web Version">
    Access ShellX.ai directly through your web browser without any installation.
    
    1. Visit [shellx.toscl.com](https://shellx.toscl.com)
    2. Sign up for a free account
    3. Start using AI-powered terminal features immediately
    
    <Note>
      The web version provides full functionality and works on any modern browser.
    </Note>
  </Accordion>
  
  <Accordion icon="android" title="Android App">
    Download the ShellX.ai Android app for mobile terminal access.
    
    1. Download from the official repository
    2. Install the APK on your Android device
    3. Grant necessary permissions for terminal access
    4. Launch and configure your AI terminal
    
    <Warning>
      Ensure your Android device has developer options enabled for full functionality.
    </Warning>
  </Accordion>
</AccordionGroup>

### First Steps

<AccordionGroup>
  <Accordion icon="user-plus" title="Create Your Account">
    Sign up for ShellX.ai to access all AI-powered features:
    
    1. Visit the registration page
    2. Enter your email and create a secure password
    3. Verify your email address
    4. Complete your profile setup
  </Accordion>
  
  <Accordion icon="terminal" title="Initialize Your Terminal">
    Set up your AI-powered terminal environment:
    
    1. Launch ShellX.ai
    2. Choose your preferred terminal theme
    3. Configure AI assistance settings
    4. Test basic commands with AI help
    
    ```bash
    # Example: Ask AI to help with file operations
    shellx help ls
    # AI will provide detailed explanations and suggestions
    ```
  </Accordion>
</AccordionGroup>
