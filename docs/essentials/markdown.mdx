## Natural Language Commands

ShellX.ai transforms natural language into terminal commands using advanced AI. Simply type what you want to do, and the AI will interpret and execute the appropriate command.

```bash
# Natural language input
shellx "show me all files in the current directory"

# AI translates to:
ls -la
```

### Basic File Operations

Use natural language for common file operations:

```bash
# List files
shellx "list all files"
shellx "show hidden files"

# Create files
shellx "create a new file called test.txt"
shellx "make a directory named projects"

# Copy and move
shellx "copy file1.txt to backup folder"
shellx "move all .js files to src directory"
```

<Tip>

The AI learns from your usage patterns and becomes more accurate over time. The more you use ShellX.ai, the better it understands your preferences.

</Tip>

### Error Explanation

When commands fail, ShellX.ai explains the error and suggests solutions:

```bash
# Error occurred
shellx "delete important.txt"
# AI Response: "File not found. Did you mean 'important_backup.txt'?"
# Suggestion: "Use 'ls' to see available files"
```

### Learning Mode

Enable learning mode to understand new commands:

```bash
# Enable learning mode
shellx --learn

# Teach new patterns
shellx "backup project" --define "tar -czf backup.tar.gz project/"
```

## Customization

### Personal Preferences

Configure ShellX.ai to match your workflow:

```bash
# Set preferred commands
shellx config set list-command "ls -la"
shellx config set editor "vim"

# Configure AI behavior
shellx config set ai-temperature 0.7
shellx config set response-length short
```

### Custom Commands

Create custom command aliases:

```bash
# Create alias
shellx alias "backup docs" "tar -czf docs_backup.tar.gz ~/Documents/"

# Use alias
shellx "backup docs"
```

<Note>
  ShellX.ai supports multiple AI models. You can switch between them using `shellx config set model <model-name>`.
</Note>
